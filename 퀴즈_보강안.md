# ChromoLab 퀴즈 보강안 (검토용)

> 만들기 전 검토 요청. 빼거나 고칠 문항/표현을 알려주세요. 승인 후 구현·배포합니다.
> 과학 정확성 규칙 준수: 후기의 "세포 전체 염색체수·핵상 변화"는 묻지 않음 / 동원체 카운팅 설명 없음 / "딸세포의 염색체"·"2가 염색체"·"감수 1분열" 표기 / 핵상 4n 미표기.

## 현재 퀴즈
무작위 시기 그림 → ① 핵상 ② 염색체 수 ③ DNA 상대량 (세포 1개 기준). 후기 등 순간 시기는 제외.

## 보강 방향
퀴즈에 **유형(카테고리) 선택**을 추가하고, 아래 새 유형들을 더한다. (체세포/감수 토글은 유지)

---

### 유형 A. 분리 대상 (그림 기반) — 오개념 M3 직격
후기 시기 그림을 보고 **무엇이 분리되는지**만 묻는다(염색체 수·핵상은 안 물음 → 정책 위배 아님).
- **후기Ⅰ**: "이 시기에 양극으로 분리되어 이동하는 것은?" → (상동염색체 / 염색분체) → **상동염색체**
  해설: 감수 1분열 후기에는 2가 염색체를 이룬 **상동염색체가 분리**돼요(염색분체 분리 아님).
- **후기Ⅱ / 체세포 후기**: "…분리되는 것은?" → **염색분체**
  해설: 자매 염색분체가 분리되어 각각이 딸세포의 염색체가 돼요.
- ⚠️ 검토 포인트: 후기 그림을 "분리 대상" 문항에 한해 쓰는 것 OK? (염색체수/핵상은 여전히 안 물음)

### 유형 B. 시기 식별 (그림 → 시기 이름)
시기 그림을 보고 "이 시기는?" 4지선다.
- 예) 2가 염색체가 적도판에 배열된 그림 → (중기 / **중기Ⅰ** / 중기Ⅱ / 후기Ⅰ)
  해설: 상동염색체가 짝지어(2가 염색체) 배열 → 중기Ⅰ.

### 유형 C. 감수분열·생식세포 다양성 (개념, 그림 없음)
- "유전자형 **AaBb**인 개체가 만들 수 있는 생식세포 **종류 수**는?" → (2 / **4** / 8) 
  해설: 두 쌍이 독립적으로 분리 → 2² = 4종류(AB·Ab·aB·ab).
- "상동염색체가 **n쌍**일 때 생식세포 종류 수는?" → (2n / **2ⁿ** / n²)
  해설: 각 쌍이 독립적으로 분리되므로 2ⁿ.
- "생식세포의 다양성을 만드는 감수분열의 핵심 사건은?" → (교차 / **상동염색체의 독립적 분리** / DNA 복제)
  ※ 교차는 본문 밖이라 정답/주요 보기에서 배제, 오답 보기로만.

### 유형 D. 수정·자손 다양성 (개념)
- "난자 **4종류**, 정자 **4종류**가 무작위로 수정될 때 가능한 (난자×정자) 조합 수는?" → (8 / **16** / 4)
- "자손의 유전적 다양성이 커지는 **두 가지** 이유는?" (복수 선택 또는 보기 중 옳은 조합)
  → **① 감수분열의 (상동염색체) 독립적 분리, ② 암수 생식세포의 무작위 수정**
- "수정으로 핵상은 어떻게 되나?" → (n→n / **n + n → 2n** / 2n→n)

### 유형 E. 체세포분열 vs 감수분열 비교 (개념)
- "체세포분열로 만들어진 딸세포는 모세포와 유전적으로?" → (다르다 / **같다**)
- "감수분열 결과 만들어지는 딸세포 수와 핵상은?" → (2개·2n / **4개·n** / 2개·n)
- "DNA 복제는 언제 일어나나?" → 체세포·감수 모두 **분열 전 S기 1회**(감수는 2번 분열하지만 복제는 1회뿐).
- "딸세포 수: 체세포분열 ___개 / 감수분열 ___개" → **2 / 4**

## 구현 방식(안)
- 퀴즈 상단에 **유형 선택 칩**(예: 시기값 / 분리·식별 / 다양성·수정 / 비교) 또는 "랜덤 전체" 추가.
- 개념 문항(C·D·E)은 그림 없이 보기형. 정답·해설은 기존처럼 한 곳(데이터)에서 관리.
- 채점·정답률·해설 콜아웃 등 기존 UI 재사용.

## 검토 요청
1. 포함할 유형: A·B·C·D·E 중 무엇무엇?
2. 유형 A에서 **후기 그림 사용** 허용 여부(분리 대상만 질문).
3. 표현·정답·보기 중 고칠 것.
